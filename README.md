# Учебный проект "Путешествие по России"


## Технологии:
* Флексбокс-вёрстка
* Grid
* Адаптивная верстка
* Подключение шрифтов
* Файловая структура организована по методологии БЭМ Nested

Проект сверстан по макету из Figma, адаптирован под любую ширину экрана.
